# Vocational Training Development Institute
This project is an assignment for the Vocational Training Development Institute, School of Applied Technology, Department of ICT. It focuses on building a Digital Onboarding System using Java, modeling the relationship between a Client and their Card.

  *******
  The Onboarding App allows clients to:
  - [x] Create a profile once the age is 18 and older.
  - [x] Update their PIN.
  - [x] Deposit & Withdraw from their card.
  - [x] Card Details & PINs are masked for security concerns.
